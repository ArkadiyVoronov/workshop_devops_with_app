# Почему мы выбрали этот стек

## Альтернативы которые рассматривали

| Решение | Почему не подошло |
|---------|-------------------|
| ELK Stack | Фокус на логи, не метрики |
| Datadog | Платный, нужен cloud аккаунт |
| New Relic | SaaS, не локально |
| Zabbix | Устаревший UI, сложно для контейнеров |

## Преимущества Prometheus + Grafana

✅ Open-source, бесплатно
✅ Cloud-native, создан для контейнеров
✅ Индустриальный стандарт (70% компаний)
✅ Локальный запуск без cloud
✅ Навык переносим в любую компанию
